# Flashcard_Group6
FlashCard Application Study Tool

TeamName: StudyGeeks

Goal: Create a Flash Card application as a study tool for students
  - Students should be able to input Q/A into a FlashCard
  - potential for automatic conversion of documents and notes into Flash cards
  - Testing of Flashcard Functionalities:
    - Randomizing of Flash Cards (Reset/Shuffle/Organize)
    - Quiz/Test Taking (Multiple Choice / Short Answer/ True-False
    - Mark for Review

## Installation
   - Fork Repo in your own Repository
   - clone repo
       - ` git clone https://github.com/(Your_User_Name)/(Repo_Name) `
   - Find/Move Cloned Repo Folder to Desired location
